1. Maak een virtual environment genaamd "naam". Dit doe je met het onderstaande commando;
$ python -m venv "naam"

2. Activeer de virtual environment in je terminal met het onderstaande commando;
$ source "naam"/Scripts/activate

3. Installeer de module "module naam" met het onderstaande commando;
$ pip install "module naam"

5. Zodra je klaar bent met het uitvoeren van het programma, deactiveer de virtual environment met het onderstaande commando;
$ deactivate
